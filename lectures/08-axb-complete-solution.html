Lecture 8 · Solving Ax = b — the complete solution

Every solution of Ax = b splits into one particular solution plus anything in the nullspace: x = xp + xn. The solution set is the nullspace shifted so it passes through xp. ▶ watch on OCW
The system. We use a short, wide matrix A = [[1, 2, 2], [2, 4, 5]] (2 equations, 3 unknowns). Rank is 2, so there is one free variable and the nullspace is a line through the origin. When b is consistent, the complete solution is that same line translated to pass through a particular solution xp — a line in R³ that does not go through the origin (unless b = 0).
Strang's own example (4-D, so we can't draw it). The lecture itself uses the 3×4 matrix A = [[1,2,2,2],[2,4,6,8],[3,6,8,10]] with b = (1,5,6) — note row 3 = row 1 + row 2, which forces the solvability condition b₁ + b₂ = b₃. Rank is 2, so there are two free variables and the solution set is a 2-D plane shifted off the origin inside R⁴. We can't plot R⁴, so the picture above uses the 2×3 cousin (a shifted line in R³) to show the same x = xp + xn geometry. The exact 4-D arithmetic is worked out below, and the four rank cases Strang ends on are at the bottom of the page.

The intuition

One particular point, then slide it. Suppose you find a single vector xp with A xp = b. Now take any xn in the nullspace (A xn = 0). Then A(xp + xn) = A xp + A xn = b + 0 = b. So every sum xp + xn solves the system, and conversely any two solutions differ by something in the nullspace. The full answer is a copy of the nullspace, picked up and dropped onto xp.

Why it usually misses the origin. The nullspace (green line) always passes through 0 because A·0 = 0. The complete solution set (red line) only passes through the origin when 0 is itself a solution — i.e. when b = 0. Toggle the homogeneous preset and watch the red line snap onto the green one through the origin; toggle back and the red line lifts away, parallel but offset by xp.

No solution looks different. The inconsistent preset keeps the same columns but moves b outside the column space. Elimination then produces an impossible row like 0 = −1; there is no xp at all, so there is no line to draw. The plot greys out and the status box shows the contradiction.

The worked math

Take the consistent case, b = [5, 11]. Augment and eliminate: 

[A | b] = [ 1  2  2 | 5 ]
          [ 2  4  5 | 11]

R2 ← R2 − 2·R1:

          [ 1  2  2 | 5 ]   ← pivot in col 1
          [ 0  0  1 | 1 ]   ← pivot in col 3

Pivot columns: 1 and 3.   Free column: 2  (no pivot)  →  one free variable x₂.

Consistency. No row of the form [0 0 0 | nonzero] appeared, so the system is consistent. (For b = [5, 9], R2 becomes [0 0 1 | −1] after the first step and a second elimination would give 0 = −1 — see the inconsistent preset.)

Particular solution — set the free variable x₂ = 0 and back-substitute: from row 2, x₃ = 1; from row 1, x₁ + 2(0) + 2(1) = 5 ⇒ x₁ = 3. So 

x_p = [ 3, 0, 1 ]ᵀ        (check: 1·3 + 2·0 + 2·1 = 5,  2·3 + 4·0 + 5·1 = 11 ✓)

Special solution (nullspace). Solve Ax = 0. Set the free variable x₂ = 1. Then row 2 gives x₃ = 0, and row 1 gives x₁ + 2(1) + 2(0) = 0 ⇒ x₁ = −2: 

s = [ −2, 1, 0 ]ᵀ         (check: 1·(−2) + 2·1 + 2·0 = 0,  2·(−2) + 4·1 + 5·0 = 0 ✓)

Complete solution. One free variable ⇒ a one-parameter family. The nullspace is the line c·s, and the complete solution is 

x = x_p + c·s = [ 3, 0, 1 ]ᵀ + c·[ −2, 1, 0 ]ᵀ ,   c ∈ ℝ

  = [ 3 − 2c ,  c ,  1 ]ᵀ

This is a line in R³ through xp = (3,0,1) with direction s = (−2,1,0). It is parallel to the nullspace line c·s through the origin but shifted off it — exactly the red/green pair you can rotate in the figure above. Set b = 0 and xp = 0: the two lines coincide and pass through the origin.

Strang's exact 4-D example, worked out

Now the real lecture matrix. A = [[1,2,2,2],[2,4,6,8],[3,6,8,10]], b = (1,5,6). Row 3 = row 1 + row 2, so elimination kills the third row and leaves the solvability test 0 = b₃ − b₂ − b₁. With (1,5,6) that is 6 − 5 − 1 = 0 ✓, so a solution exists. 

[A | b] = [ 1 2 2 2 | 1 ]        eliminate         [ 1 2 2 2 | 1 ]   ← pivot col 1
          [ 2 4 6 8 | 5 ]   ───────────────▶      [ 0 0 2 4 | 3 ]   ← pivot col 3
          [ 3 6 8 10| 6 ]                          [ 0 0 0 0 | 0 ]   ← 0 = b₃−b₂−b₁ ✓

Pivot columns: 1, 3.   Free columns: 2, 4   →   two free variables x₂, x₄.

Particular solution — set both free variables x₂ = x₄ = 0. Row 2 gives 2x₃ = 3 ⇒ x₃ = 3/2; row 1 gives x₁ + 2x₃ = 1 ⇒ x₁ = 1 − 3 = −2: 

x_p = ( −2, 0, 3/2, 0 )ᵀ      (Strang's "minus two, zero, three halves, zero")

Two special solutions (solve Ax = 0, one free variable = 1 at a time). The reduced rows are x₁ + 2x₂ + 2x₄ = 0 and x₃ + 2x₄ = 0: 

x₂ = 1, x₄ = 0  →  s₁ = ( −2, 1,  0, 0 )ᵀ
x₂ = 0, x₄ = 1  →  s₂ = (  2, 0, −2, 1 )ᵀ        (Strang: "two minus four and two gives zero, check")

Complete solution. Two free variables ⇒ a two-parameter family — a 2-D plane, shifted off the origin, living inside R⁴: 

x = x_p + c₁·s₁ + c₂·s₂
  = ( −2, 0, 3/2, 0 )ᵀ + c₁·( −2, 1, 0, 0 )ᵀ + c₂·( 2, 0, −2, 1 )ᵀ,   c₁, c₂ ∈ ℝ

xp is one point; the two s's span the nullspace plane through the origin; adding them lifts that plane up to xp. As Strang puts it, "it's like a flat thing, like a subspace, but it's been shifted away from the origin — it doesn't contain zero." It is exactly the R³ line picture at the top of this page, with one extra dimension of freedom.

The big picture — what the rank tells you

Strang closes the lecture with the line that "summarizes the whole lecture": the rank r tells you everything about the number of solutions — everything except the exact entries.
