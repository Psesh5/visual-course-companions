The intuition

Orthogonal = perpendicular = right angle. The dot product measures how much two vectors point the same way. When you tilt a and b until the dot product passes through zero, the arrows form a perfect right angle. At that instant the triangle with legs a and b and hypotenuse a+b is a right triangle — and the Pythagorean theorem is the statement ‖a+b‖² = ‖a‖² + ‖b‖². Try Figure 1's "Perpendicular" preset and watch all three squared lengths line up.

Orthogonal subspaces. Two subspaces are orthogonal when every vector in one is perpendicular to every vector in the other. A plane through the origin and the line along its normal are the cleanest example (Figure 2): no matter which vector you pick in the plane, it makes a right angle with the normal direction. The orthogonal complement V⟂ collects all vectors perpendicular to V; in R³ a plane (dim 2) pairs with a line (dim 1), and 2 + 1 = 3.

The big payoff — the four subspaces. Strang's "big picture": a matrix A with n columns and m rows splits its two homes perpendicularly. Inside Rⁿ, the row space and the nullspace are orthogonal complements (r + (n−r) = n). Inside Rᵐ, the column space and the left nullspace are orthogonal complements (r + (m−r) = m). Why? If Ax = 0, then each row of A dotted with x is zero — so x is perpendicular to every row, hence to the whole row space. Figure 3 shows this for a concrete rank-2 matrix.

The worked math

1 · Dot product and the definition of orthogonal. For a, b ∈ Rⁿ,

a · b = aᵀb = a₁b₁ + a₂b₂ + ··· + aₙbₙ = ‖a‖ ‖b‖ cos θ.

The length is ‖a‖ = √(a·a). Since ‖a‖, ‖b‖ > 0 for nonzero vectors, a · b = 0 exactly when cos θ = 0, i.e. θ = 90°. That is the definition: a ⟂ b ⇔ a·b = 0. (By convention the zero vector is orthogonal to everything, since 0·b = 0.)

2 · The Pythagorean derivation. Expand the squared length of the hypotenuse using ‖v‖² = v·v and the distributive law:

‖a + b‖² = (a+b)·(a+b) = a·a + 2(a·b) + b·b = ‖a‖² + 2(a·b) + ‖b‖².

The cross term 2(a·b) vanishes iff a·b = 0. So ‖a + b‖² = ‖a‖² + ‖b‖² holds exactly when a ⟂ b — Pythagoras and orthogonality are the same statement. Figure 1's status box reports all three squared lengths so you can see the equation balance.

3 · Subspaces and the orthogonal-complement dimension law. Subspaces V, W ⊆ Rⁿ are orthogonal if v·w = 0 for all v ∈ V, w ∈ W. The orthogonal complement is

V⟂ = { x ∈ Rⁿ : x · v = 0 for every v ∈ V }.

It is itself a subspace, and it captures all perpendicular directions, giving the dimension relation

dim V + dim V⟂ = n.

For the four subspaces of an m×n matrix of rank r this becomes the fundamental theorem of linear algebra (part 2):

In Rⁿ: row space (dim r) ⟂ nullspace (dim n−r)  ⇒  r + (n−r) = n
In Rᵐ: column space (dim r) ⟂ left nullspace (dim m−r)  ⇒  r + (m−r) = m

The row-space/nullspace orthogonality is immediate: Ax = 0 says every row rᵢ satisfies rᵢ · x = 0, so x is orthogonal to each row and therefore to their entire span, the row space. The same argument on Aᵀ gives column space ⟂ left nullspace. Figure 3 confirms the first pair numerically for a rank-2 example.

4 · The matrix AᵀA (coming attraction). Strang closes by previewing the matrix behind least squares. When Ax = b has no solution (too many noisy equations), the good equation to solve instead is

AᵀA x̂ = Aᵀb.

For any m×n matrix A, the product AᵀA is n×n (square) and symmetric, since (AᵀA)ᵀ = Aᵀ(Aᵀ)ᵀ = AᵀA. The key facts are

N(AᵀA) = N(A)  and  rank(AᵀA) = rank(A).

(Reason: if AᵀAx = 0 then xᵀAᵀAx = ‖Ax‖² = 0, forcing Ax = 0; the converse is immediate.) So AᵀA is invertible exactly when A has independent columns. Strang's two examples (Figure 4): with A = [[1,1],[1,2],[1,5]] (independent columns) we get AᵀA = [[3,8],[8,30]], determinant 90 − 64 = 26 ≠ 0, invertible; but with a rank-1 A the product is rank-1 and singular (determinant 0).
