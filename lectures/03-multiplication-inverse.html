Lecture 3 · Matrix multiplication and inverse matrices

One idea, three views: Ax is a combination of A's columns, AB is built one column at a time, and A⁻¹ is the transform that undoes A. ▶ watch on OCW

1 · Ax as a combination of columns

Drag the sliders for x₁, x₂, x₃. The arrows chain tip-to-tail: x₁·col₁ + x₂·col₂ + x₃·col₃. Where the chain lands is Ax (the red arrow). Solving Ax = b means choosing the x's so the chain lands on b.
x₁ = 1 x₂ = 1 x₃ = 1

2 · Matrix multiplication AB, column by column

The key fact: column j of AB equals A times column j of B. So computing AB is just two (or n) separate "A times a column" problems. Use the buttons to step through the worked example A·B with A = [[1,2],[3,4]], B = [[5,6],[7,8]].

2b · The 4th way: columns × rows (outer products)

Strang's favorite surprise: AB is also a sum of (column of A)(row of B) pieces. Each piece — a column times a row — is a full-sized matrix, but a very special one: rank 1. His example is a single column times a single row, (2,3,4)·(1,6). Toggle the row/column entries and watch every row of the product stay a multiple of the row (1,6) and every column stay a multiple of (2,3,4) — so the whole answer collapses onto two lines.
col = (2, 3, 4) row = (1, 6)

3 · The inverse undoes the transformation

A matrix is a transformation of the plane. Below, the unit square + grid (faint) is pushed through A = [[1,3],[2,7]] to the colored shape. Then we apply A⁻¹ = [[7,−3],[−2,1]], which carries every point back to where it started — that is what "inverse" means: A⁻¹A = I.
t = 0.00

3b · When the inverse fails — the singular matrix

"Best to start with an example," Strang says, and writes A = [[1,2],[3,6]] — the matrix that has no inverse. Why? Its two columns (1,3) and (2,6) lie on the same line, so every combination x₁·col₁ + x₂·col₂ stays on that line — A squashes the whole plane onto it and can never be undone. His clinching reason: there's a nonzero x with Ax = 0. Drag the sliders to x = (2, −1) and the red output arrow vanishes to the origin — 2·(1,3) − 1·(2,6) = (0,0). If A⁻¹ existed, A⁻¹(Ax)=x would force x=0, a contradiction. So no inverse.
x₁ = 2 x₂ = -1

What you're seeing

Columns, not rows. The slickest way to read Ax is not "row · x" but as a weighted sum of A's columns. If A has columns c₁, c₂, c₃ then Ax = x₁c₁ + x₂c₂ + x₃c₃. Figure 1 builds exactly that sum with arrows. The set of all reachable points (every choice of x) is the column space of A.

AB is many Ax's. Multiply A by B by handing A each column of B in turn. Column 1 of the answer is A·(column 1 of B), column 2 is A·(column 2 of B), and so on. Figure 2 shows the arithmetic for each column separately, then stacks them into the full product.

Inverse = reverse. Geometrically A stretches and shears the grid (Figure 3); A⁻¹ is the unique transform that puts every point back. Algebraically A⁻¹A = AA⁻¹ = I. A matrix has an inverse exactly when it doesn't collapse space — i.e. det A ≠ 0 and the columns are independent. Here det A = 1·7 − 3·2 = 1 ≠ 0, so A is invertible.

The math, step by step

(a) Ax as a column combination.

A = | 2  1  0 |        x = (x₁, x₂, x₃)
    | 1  3  1 |
    | 1  1  2 |

Ax = x₁·| 2 | + x₂·| 1 | + x₃·| 0 |
        | 1 |      | 3 |      | 1 |
        | 1 |      | 1 |      | 2 |

   = ( 2x₁ + 1x₂ + 0x₃ ,
       1x₁ + 3x₂ + 1x₃ ,
       1x₁ + 1x₂ + 2x₃ )

Each slider in Figure 1 scales one column; the tip-to-tail sum of the scaled columns is the output vector.

(b) AB one column at a time. With A = [[1,2],[3,4]] and B = [[5,6],[7,8]]:

Column 1 of AB = A · (column 1 of B) = A · (5, 7)
   = 5·(1,3) + 7·(2,4) = (5+14, 15+28) = (19, 43)

Column 2 of AB = A · (column 2 of B) = A · (6, 8)
   = 6·(1,3) + 8·(2,4) = (6+16, 18+32) = (22, 50)

So   AB = | 19  22 |
          | 43  50 |

(Check one entry the "row · column" way: row 1 of A dotted with column 1 of B = 1·5 + 2·7 = 19. Same answer — the column view just bundles those dot products together.)

(b2) The 4th way — columns × rows (outer products). A whole column of A times a whole row of B is a full matrix of rank 1. Strang's example, one column (2,3,4) times one row (1,6):

| 2 |              | 2·1  2·6 |   | 2  12 |
| 3 | · ( 1  6 ) = | 3·1  3·6 | = | 3  18 |
| 4 |              | 4·1  4·6 |   | 4  24 |

Every COLUMN is a multiple of (2,3,4):   col₂ = 6·col₁
Every ROW    is a multiple of (1,6):      row₂ = 1.5·row₁, row₃ = 2·row₁
→ column space = the line through (2,3,4);  row space = the line through (1,6).  (rank 1)

General rule (4th way):   AB = Σ_k (column k of A)·(row k of B)

This is the same product as the column-by-column view — just sliced the other way. Adding up these rank-1 pieces over all k rebuilds the full AB.

(c) Gauss–Jordan inverse of A = [[1,3],[2,7]]. Augment with the identity and row-reduce until the left block is I; the right block becomes A⁻¹.

[ A | I ] =  | 1  3 | 1  0 |
             | 2  7 | 0  1 |

R2 ← R2 − 2·R1   (clear the 2 below the pivot)
             | 1  3 | 1  0 |
             | 0  1 |-2  1 |

R1 ← R1 − 3·R2   (clear the 3 above the second pivot)
             | 1  0 | 7 -3 |
             | 0  1 |-2  1 |

Left block is now I, so the right block is A⁻¹:

   A⁻¹ = |  7  -3 |
         | -2   1 |

Check:  A·A⁻¹ = | 1 3 |·|  7 -3 | = | 1·7+3·(-2)  1·(-3)+3·1 | = | 1 0 |  = I  ✓
                | 2 7 | | -2  1 |   | 2·7+7·(-2)  2·(-3)+7·1 |   | 0 1 |

The pivots were both 1 (no division needed), so the inverse came out with clean integers. In general you divide each pivot row through to make the pivot 1 — and if any pivot is 0 you cannot finish, which is the algebraic signal that A has no inverse.

(d) Why A = [[1,3],[2,6]] has no inverse. (Strang's "one three two six" — columns col₁ = (1,3) and col₂ = (2,6).)

A = | 1  2 |     det A = 1·6 − 2·3 = 0                    ← determinant test
    | 3  6 |     col₂ = (2,6) = 2·(1,3) = 2·col₁          ← columns on one line

Best reason (Strang's favorite): there is a NONZERO x with Ax = 0.
Since col₂ = 2·col₁, we have 2·col₁ − col₂ = 0, i.e. x = (2, −1):

   A·(2, −1) = 2·col₁ − 1·col₂ = 2·(1,3) − (2,6) = (2−2, 6−6) = (0, 0)  ✓

If A⁻¹ existed:   x = I·x = (A⁻¹A)·x = A⁻¹·(Ax) = A⁻¹·0 = 0,
but x = (2,−1) ≠ 0 — contradiction. A matrix that sends some nonzero x to 0 cannot be inverted.

Geometrically: A flattens the whole plane onto the single line through (1,3), so distinct points pile onto the same image and there is no way back. The figure above lets you slide to x = (2, −1) and watch the output arrow collapse to the origin.
