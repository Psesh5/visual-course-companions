Lecture 1 · The geometry of linear equations

Strang's central idea: a system Ax = b has two pictures — the row picture (planes meeting) and the column picture (a combination of columns reaching b). ▶ watch on OCW
The explorer: type any 3×3 system below and press Plot. The left panel draws one plane per equation (the row picture); the right panel chains the columns tip-to-tail (the column picture). The status box tells you whether you have one, no, or infinitely many solutions — and why.

Start in 2D — two lines, two columns

Strang opens with the 2×2 case as "the right place to start": 2x − y = 0 and −x + 2y = 3, meeting at (1, 2). Edit the numbers below. Left = row picture (two lines crossing); right = column picture (combine the two columns x·col₁ + y·col₂ tip-to-tail to reach b). Tick show all combinations to watch those two columns' combinations tile the whole plane — Strang's "what are all the combinations?" question. (Make the columns dependent and the grid collapses to a single line — that's the singular case.)

Matrix × vector = a combination of columns

The matrix form is Ax = b, and the key idea Strang ends on: Ax is a combination of the columns of A. His example A = [[2,5],[1,3]], x = (1,2) gives 1·(2,1) + 2·(5,3) = (12,7). Slide x₁, x₂ and watch the tip-to-tail combination move.

What you're seeing

Row picture (left). Each equation a·x + b·y + c·z = d is a flat plane. An equation with a missing variable (like 2x − y = 0, no z) is a vertical plane — a wall that runs straight up the z-axis. Rotate until you see where all three planes cross. For a normal (invertible) system that's a single point — the solution.

Column picture (right). Rewrite the system as x·col₁ + y·col₂ + z·col₃ = b. Solving asks: what amount of each column vector, added tip-to-tail, lands exactly on b? The colored arrows show that chain; the red arrow is b.

The singular cases

Load the two singular presets. When the columns are dependent (one is a combination of the others) they no longer fill 3D space — they span only a plane. Then either b is off that plane (no solution: the planes enclose a tunnel with no common point) or b lies on it (infinitely many: the planes share a whole line). This is what "det = 0" looks like.

Tilt the bottom plane — watch the solution slide

The lecture system is fixed at the top two equations 2x − y = 0 and −x + 2y − z = −1; the slider changes only the z-coefficient of the third equation −3y + 4z = 4. At the lecture value 4 the planes meet at (0, 0, 1). Slide it and watch the red point slide off the z-axis — proof that changing the coefficient of a nonzero variable (here z = 1) genuinely moves the solution in all three coordinates, while the earlier x/y changes did not.
