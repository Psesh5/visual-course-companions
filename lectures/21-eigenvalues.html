Lecture 21 · Eigenvalues and eigenvectors

For most vectors x, multiplying by A turns them in a new direction. Eigenvectors are the special directions that don't turn: Ax = λx. The matrix only stretches them by the scalar eigenvalue λ. ▶ watch on OCW
The picture: we send a whole ring of unit vectors v (light blue) through the matrix and draw every image A·v (faint orange). Almost every arrow lands pointing somewhere new. But along the eigen-directions the image stays on the same line as the input — only longer or shorter. Those are highlighted; their length factor is the eigenvalue λ.

The intuition

A matrix is a transformation: feed it a vector, it spits out another vector. Watch the ring above — a circle of inputs becomes a tilted ellipse of outputs, and the arrows generally swing to a new angle. The eigenvectors are the few input directions that survive the transformation pointing the same way (or exactly opposite, if λ is negative). On those lines A acts like a plain number: it multiplies by λ. That is the whole content of Ax = λx.

For Strang's symmetric [[3,1],[1,3]] the eigen-directions are (1, 1) (the 45° line, stretched by λ = 4) and (−1, 1) (the anti-diagonal, stretched by λ = 2). Symmetric matrices always have perpendicular eigenvectors and real eigenvalues — notice those two highlighted lines meet at a right angle, and (1,1)·(−1,1) = 0. As Strang says, the instant he sees (1,1) and (−1,1) his mind takes the dot product and gets zero.

When there are no real eigen-directions. Try Strang's rotation preset Q = [[0,−1],[1,0]], the matrix that turns every vector by 90°. Every single arrow rotates — nothing stays on its own line, so there is no real eigenvector. The trace says the eigenvalues add to 0 and the determinant says they multiply to +1; the only way to satisfy both is the complex pair λ = i and λ = −i. Strang notes this happens because Q is anti-symmetric (Qᵀ = −Q), the extreme opposite of a symmetric matrix — anti-symmetric matrices have pure-imaginary eigenvalues. Worked out: det(Q − λI) = (−λ)(−λ) − (−1)(1) = λ² + 1 = 0 ⇒ λ = ±i.

Strang's projection matrix: eigenvalues 1 and 0

Strang opens the lecture with a projection matrix P onto a plane (before any determinants). He asks: which vectors get projected back to the same direction? Two families:

  • Any vector already in the plane doesn't move — Px = x, so λ = 1 (a whole plane of eigenvectors).
  • Any vector perpendicular to the plane projects to nothing — Px = 0 = 0·x, so λ = 0 (the null space). A projection is singular, and a singular matrix always has λ = 0.

The 2D analog you can plot above is the preset [[1,0],[0,0]], projection onto the x-axis. Watch the ring of inputs collapse onto the horizontal line: vectors on the x-axis (1,0) stay put (λ = 1), and the vertical vector (0,1) is flattened to the origin (λ = 0). trace = 1 = 1 + 0 and det = 0 = 1 · 0 ✓.

Strang's permutation matrix and the "+3I" shift

The permutation preset A = [[0,1],[1,0]] swaps the two components of a vector. Spot the eigenvectors by eye: (1,1) is unchanged when you swap its entries, so λ = 1; and (−1,1) swaps into (1,−1) = −(−1,1), so λ = −1. The trace is 0 = 1 + (−1) — once you know one eigenvalue of a 2×2 matrix, the trace hands you the other.

The shift trick. Strang's symmetric example is exactly this permutation matrix plus 3I: [[0,1],[1,0]] + 3I = [[3,1],[1,3]]. Adding 3I leaves every eigenvector unchanged and bumps every eigenvalue up by 3, so λ = 1, −1 become λ = 4, 2. (Careful, he warns: this works only because we added a multiple of I. For a general A + B the eigenvalues do not add — the matrices usually have different eigenvectors.)

Two invariants you can read off instantly

Before solving anything, two numbers already pin down the eigenvalues of a 2×2 matrix:

  • trace = a + d = λ₁ + λ₂  (the eigenvalues add to the diagonal sum)
  • det = ad − bc = λ₁ · λ₂  (the eigenvalues multiply to the determinant)

These come directly from the characteristic equation λ² − (trace)·λ + (det) = 0 (sum & product of roots). They are a great sanity check and are shown live in the status box.

Worked math — det(A − λI) = 0 for [[3,1],[1,3]]

Following Strang exactly: subtract λ from the diagonal and set the determinant to zero: 

A − λI = [ 3−λ    1   ]
         [  1    3−λ  ]

det(A − λI) = (3−λ)(3−λ) − (1)(1)
            = λ² − 6λ + 9 − 1
            = λ² − 6λ + 8          ← 6 is the TRACE, 8 is the DETERMINANT
            = (λ − 4)(λ − 2) = 0

  ⇒  λ₁ = 4 ,  λ₂ = 2       (the characteristic equation)

Eigenvector for λ = 4. Make A − 4I singular and find its null space: 

(A − 4I) = [ −1   1 ]      [ −1   1 ][x₁]   [0]
           [  1  −1 ]  ⇒   [  1  −1 ][x₂] = [0]

  row says  −x₁ + x₂ = 0  ⇒  x₂ = x₁  ⇒  x = (1, 1)

Eigenvector for λ = 2. Make A − 2I singular and find its null space: 

(A − 2I) = [ 1   1 ]      row says  x₁ + x₂ = 0
           [ 1   1 ]  ⇒   ⇒  x₂ = −x₁  ⇒  x = (−1, 1)

Check. A·(1,1) = (3·1 + 1·1, 1·1 + 3·1) = (4, 4) = 4·(1,1) ✓   and A·(−1,1) = (−3+1, −1+3) = (−2, 2) = 2·(−1,1) ✓.

Invariant check. trace = 3 + 3 = 6 = 4 + 2 = λ₁ + λ₂ ✓   and det = 3·3 − 1·1 = 8 = 4·2 = λ₁·λ₂ ✓.

The bad case — a repeated eigenvalue with too few eigenvectors

Strang ends with the cautionary example A = [[3,1],[0,3]] (the degenerate preset). It is triangular, so the eigenvalues are read straight off the diagonal: λ = 3, repeated twice (det(A − λI) = (3−λ)(3−λ) = 0). The trouble is the eigenvectors: 

(A − 3I) = [ 0   1 ]   null space:  0·x₁ + 1·x₂ = 0  ⇒  x₂ = 0
           [ 0   0 ]              ⇒  the ONLY eigenvector direction is x = (1, 0)

There is no second independent eigenvector — a 2×2 matrix with only one line of eigenvectors. Strang calls this a degenerate matrix: a repeated eigenvalue opens the door to a shortage of eigenvectors, and "a repeated lambda is the source of all trouble in 18.06." In the plot you'll see only one highlighted eigen-direction even though the algebra reports λ = 3 twice.
