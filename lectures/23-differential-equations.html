The intuition: eigenvalues steer the flow

A differential equation du/dt = Au says: the velocity of the point u is the matrix A applied to its current position. That is exactly the gray vector field — drop a particle anywhere and it rides the arrows. The colored trajectories are what you get by following those arrows in time.

The magic is that the messy curved flow is really just a blend of pure exponentials along the eigenvectors. Start exactly on an eigenvector x and you stay on that line forever, scaling like eλt — that is the dashed straight-line solution. A real eigenvalue λ<0 pulls the line inward (decay); λ>0 pushes it outward (growth); λ=0 freezes it (a line of steady states). Every other start is a mix of the two, so the long-run behavior is governed by whichever eigenvalue has the largest real part.

  • All Re(λ) < 0 → every term eλt decays → stable, flow spirals/slides into the origin.
  • Some λ = 0 (rest < 0) → one term is constant → steady state, flow settles onto the λ=0 eigenvector line.
  • Any Re(λ) > 0 → that term explodes → unstable, trajectories fly off.
  • Complex λ = a ± bi → eat(cos bt ± i sin bt) → rotation (the b) times growth/decay (the a) → a spiral.

The worked math

1. Diagonalize. Write A = SΛS⁻¹, where the columns of S are the eigenvectors x₁, x₂ and Λ = diag(λ₁, λ₂). Substituting u = Sv turns the coupled system into dv/dt = Λv, which is just two uncoupled scalar equations dvᵢ/dt = λᵢ vᵢ, each solved by vᵢ(t) = eλᵢt vᵢ(0).

2. General solution. Transforming back,   u(t) = c₁ eλ₁t x₁ + c₂ eλ₂t x₂,   where c = S⁻¹u(0) are the coordinates of the start in the eigenvector basis.

3. The matrix exponential — the power series. "There's a power series for exponentials," says Strang. Borrow the scalar Taylor series ex = Σ xⁿ/n! = 1 + x + x²/2! + x³/3! + … and just put a matrix in the exponent: eAt = I + At + (At)²/2! + (At)³/3! + … = Σ (At)ⁿ/n!. This is the definition, and it is better-behaved than the scalar geometric series — dividing by the factorials n! makes it always converge, for any matrix A and any t.

Why it equals S eΛt S⁻¹. Substitute A = SΛS⁻¹. Each power telescopes because the inner factors cancel: A² = SΛS⁻¹·SΛS⁻¹ = SΛ²S⁻¹, and generally Aⁿ = SΛⁿS⁻¹. Writing I = S I S⁻¹ and pulling the common S … S⁻¹ out of every term:
eAt = S(I + Λt + Λ²t²/2! + …)S⁻¹ = S eΛt S⁻¹, where eΛt = diag(eλ₁t, eλ₂t) is just the scalar exponentials down the diagonal. Then u(t) = eAt u(0) reproduces the eigenvector sum in step 2. (Side note from lecture: the same trick on I + At + (At)² + … — no factorials — sums to (I − At)⁻¹, the geometric series, but that one only converges when the eigenvalues of At are below 1. The factorial version never fails. The whole construction assumes A is diagonalizable — degenerate matrices without a full set of eigenvectors have no S⁻¹.)

4. Stability. Since S and S⁻¹ don't move with t, the fate of eAt is the fate of eΛt: eAt → 0 (so u→0 for every start) iff every Re(λᵢ) < 0 — the eigenvalues live in the left half of the complex plane. A zero eigenvalue leaves a surviving constant term (steady state); any positive real part makes eAt blow up. For a complex λ = a ± bi only the real part a decides growth: |e(a+bi)t| = eat, since |cos bt + i sin bt| = 1 just rotates on the unit circle. (Contrast: matrix powers Aᵏ → 0 need |λ| < 1 — eigenvalues inside the unit circle, not the left half-plane.)

5. The 2×2 shortcut. For A = [[a,b],[c,d]] you can read stability off the matrix without finding the eigenvalues. Both eigenvalues have negative real part iff trace = a+d < 0 and det = ad−bc > 0. Reason: trace = λ₁+λ₂ and det = λ₁λ₂; two negatives give a negative sum and a positive product. A negative trace alone is not enough — Strang's counterexample [[−2,0],[0,1]] has trace −1 < 0 but det = −2 < 0, a λ = +1 hiding inside, so it blows up. (Reversing every sign of A flips the sign of each eigenvalue — turning the stable example into a blow-up.)

Worked example — Strang's matrix on the board

Strang's example (load the first preset) is A = [[−1, 2], [1, −2]] with start u(0) = (1, 0) — "everything is in u₁ at time zero." He reads the eigenvalues straight off the matrix: the second column is −2 times the first, so A is singular ⇒ λ₁ = 0. The trace is −1 + (−2) = −3, and eigenvalues sum to the trace, so λ₂ = −3. (Check: det(A − λI) = λ² + 3λ = λ(λ+3).)

  • λ₁ = 0: Ax = 0 ⇒ −x + 2y = 0 ⇒ x₁ = (2, 1).
  • λ₂ = −3: (A+3I)x = 0 with [[2,2],[1,1]] ⇒ x + y = 0 ⇒ x₂ = (1, −1).

So u(t) = c₁ e0·t(2,1) + c₂ e−3t(1,−1) = c₁(2,1) + c₂ e−3t(1,−1). Matching u(0) = (1,0) gives 2c₁ + c₂ = 1 and c₁ − c₂ = 0, so c₁ = c₂ = 1/3 (Strang spots this by eye: equal weights give (3,0), scale by a third). The e−3t term decays away; the constant term survives. As t → ∞ every trajectory glides onto the λ=0 eigenvector line and stops at the steady state u(∞) = ⅓·(2,1) = (2/3, 1/3) — "two-thirds of one-third."

Strang's final example — turning one 2nd-order ODE into a 2×2 system

The whole machine above eats a first-order system du/dt = Au. But most physics gives you a single higher-order equation — a mass on a spring with damping is y″ + b y′ + K y = 0. Strang's trick (the same one he used for Fibonacci) is to stack the unknown and its derivative into a vector and add one trivial equation, converting the scalar 2nd-order ODE into a 2×2 first-order system that the eigenvalue method solves directly.

Let the state vector be u = (y′, y). Then:

  • The ODE itself says y″ = −b y′ − K y  → first row of A.
  • The trivial identity y′ = y′  → second row of A (just a 1 and a 0).

Stacked:  d/dt (y′, y) = [[−b, −K], [1, 0]] · (y′, y),  so A = [[−b, −K], [1, 0]]. Its eigenvalues are exactly the roots of the equation's own characteristic polynomial λ² + bλ + K = 0 — the eigenvalues "come out in a natural way connected to the differential equation." (A 5th-order equation becomes a 5×5 with the coefficients along the top row and ones on the subdiagonal — a companion matrix.) Play with the damping b and stiffness K below and watch the phase portrait of (y′, y): under-damped springs spiral in, over-damped ones slide straight to rest, and b < 0 (negative damping = energy pumped in) blows up.
