The intuition — a matrix you can draw

Forget formulas for a second. A graph is just dots (nodes) joined by arrows (edges). The incidence matrix A is the bookkeeping: one row per edge, one column per node. In the row for the edge that runs from node i to node j, put −1 in column i (where it leaves) and +1 in column j (where it arrives). Everything else is zero. That single rule encodes the whole picture.

Now give every node a number x — think voltage / potential / elevation. Multiplying Ax subtracts the start value from the end value for each edge, so Ax is the list of potential differences (voltage drops) along the edges. Two facts fall out immediately, and they are the heart of the lecture:

  • Raise everything by the same amount and nothing changes. If x = (c,c,c,c), every difference is c − c = 0, so Ax = 0. The constant vector (1,1,1,1) is the nullspace: potential is only defined up to a reference level (pick a ground).
  • Go around a loop and you come back to where you started. Add the drops around any closed loop and they cancel — that is Kirchhoff's Voltage Law. The loops live in the left nullspace of A (the solutions of Aᵀy = 0 are currents that balance at every node — Kirchhoff's Current Law).

The four subspaces, made concrete

A is 5×4 (m = 5 edges, n = 4 nodes). Its rank is r = n − 1 = 3 for a connected graph — the columns add to zero, so they are dependent and one is redundant. Everything else follows from r = 3:

Column space C(A) — in ℝ⁵
dim 3
The voltage-drop vectors that actually can occur. A vector b of edge drops is achievable iff its sum around every loop is zero.
Nullspace N(A) — in ℝ⁴
dim 1
Spanned by (1,1,1,1). Potentials are defined only up to a constant. n − r = 4 − 3 = 1.
Row space C(Aᵀ) — in ℝ⁴
dim 3
Spanned by 3 independent edges (a spanning tree). Orthogonal to (1,1,1,1).
Left nullspace N(Aᵀ) — in ℝ⁵
dim 2
The independent loops. m − r = 5 − 3 = 2 = Kirchhoff currents around the two independent cycles.

Euler's formula ties it together. Number of independent loops = edges − nodes + 1 = 5 − 4 + 1 = 2 for a connected graph. That is exactly m − r, the dimension of the left nullspace. A spanning tree uses n − 1 = 3 edges (no loops); each of the remaining m − (n−1) = 2 edges closes exactly one independent loop.

2 · The nullspace is the constants

Solve Ax = 0. Edge 1→2 forces x₂ = x₁; edge 2→3 forces x₃ = x₂; edge 1→4 forces x₄ = x₁. Following the edges through the connected graph, every node equals every other: x₁ = x₂ = x₃ = x₄. So

N(A) = span{ (1, 1, 1, 1) },   dim N(A) = n − r = 4 − 3 = 1.

Physically: only differences of potential are observable. We fix the indeterminacy by grounding a node, e.g. setting x₁ = 0.

3 · Independent loops (Euler)

A connected graph on n = 4 nodes has a spanning tree with n − 1 = 3 edges and no loops. We have m = 5 edges, so the 5 − 3 = 2 extra edges create 2 independent loops. By Euler, loops = m − n + 1 = 5 − 4 + 1 = 2 = dim N(Aᵀ). These two loop vectors y satisfy Aᵀy = 0 — currents that circulate without piling up charge at any node (Kirchhoff's Current Law).

4 · The physics: Ax = voltage drops, Aᵀy = 0 = KCL

Pick potentials x on the nodes. Each edge's entry of Ax is xend − xstart — the voltage drop driving current across that edge. With the default x = (0, 1, 4, 3):

Change x by a constant and these drops are unchanged (nullspace). On the other side, Aᵀ is 4×5: row i sums the edges touching node i with sign. So Aᵀy = 0 says, at every node, current in = current out — exactly Kirchhoff's Current Law. Together Ax (KVL / potential differences) and Aᵀy = 0 (KCL) are the full circuit model living inside one little matrix.

5 · Aᵀ and Kirchhoff's Current Law, node by node

Transpose A to get the 4×5 matrix Aᵀ (columns become rows). Now each row of Aᵀ is a node, and it collects every edge current touching that node with sign: +1 if the edge arrives, −1 if it leaves. Setting Aᵀy = 0 gives one balance equation per node — Strang's exact four:

Each says current in = current out: charge never piles up. The first row, −y₁ − y₃ − y₄ = 0, is node 1 — edges 1, 3, 4 all leave it, so their currents must sum to zero.

6 · The left nullspace = loop currents (try them)

A solution y of Aᵀy = 0 is a set of currents that circulate forever without a source — and the simplest ones run around a loop. The space is dim = m − r = 5 − 3 = 2, so there are two independent loops. Send one amp around each and read off the current vector; flip to the big outer loop to see it is just their sum (so not independent — exactly why the basis has only two vectors).

7 · The whole cycle: e = Ax, y = Ce, Aᵀy = f

Strang closes the lecture by snapping the three pieces into one loop — "the most important model in applied math." Read it top to bottom:

Step 1 · potentials → drops
e = A x
Node potentials x map to the voltage drops e on the edges (what the live figure at the top computes).
Step 2 · Ohm's Law
y = C e
Each drop drives a current through the edge's conductance C (= 1/resistance). C is a diagonal matrix of physical constants.
Step 3 · KCL (balance)
Aᵀy = f
Currents balance at every node. f is the outside source (current pumped in/out); with no source it is the Aᵀy = 0 above.

Substitute up the chain — y = Ce = C(Ax) into Aᵀy = f — and the three steps collapse into one equation in the unknown potentials:

Aᵀ C A x = f

That matrix AᵀCA is the heart of it. Strang's parting fact: AᵀA (and AᵀCA) is always symmetric — (AᵀCA)ᵀ = Aᵀ Cᵀ A = AᵀCA since C is diagonal. Add a current source f (or batteries in the edges) and you have the equilibrium equations of an entire electrical circuit — the same AᵀCA pattern shows up in structures, hydraulics, and least squares.
